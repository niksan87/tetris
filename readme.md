# Tetris game

A simple implementation of the classic game.

## Technologies

-   Typescript
-   Pixi
-   Webpack

### Prerequisites

```
npm, node
```

### Installing

```
npm i
```

And chose one of the scripts available:

```
npm run build-watch
npm run build-dev
npm run build-prod
npm run clean-build
npm run clean-all
npm run types-check
npm run types-check:watch
```
